Memo Editor

Memo Editor enables to write and edit paragraphs. Memo is stored as an attribute by clicking "add" button in Editor page.

Memo Editor

Figure 1. Page of Memo Editor

Fields

Memo (editable)
Memo is written in this box. If quotation mark is required in sentences, single quotation mark shall be used. Double quotation mark is not allowed.
