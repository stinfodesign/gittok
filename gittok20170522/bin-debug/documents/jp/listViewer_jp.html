List Viewer
リスト表示

リストとはこの場合，実世界に存在する現象を抽象化して示す，一次元の離散的な値をとるproxy属性の一覧表，つまりリストです．これは，実世界に起きる現象を１次元の空間にマッピングした結果と捉えることができるので，一次元地図ということもできます．GittokのList Viewer では，地物のproxy属性をキーワードとして，地物の様々な属性を表示することもできます．

List Viewer

図１．Cartographer のページ，リスト表示 (List Viewer) のページ及び，info. page

List Viewer では，以下の手順で情報の表示を行います．

１．情報表示を行う kit の選択
２．表示のルールを示すリストスキーマの選択
３．表示を行う地物型の選択．これによって，poxy value list に地物インスタンスの一覧が表示される．
４．info. pageやgeometry page を使って表示を行う属性の選択．これによって，地物インスタンスの属性が表示される．

Fields

Kit
リスト表示の対象となる kit のファイル名が表示されます．

List schema
kitをのリスト表示ルールを示すリストスキーマの名前がここに示されます．

Feature type list
Kitの中にある，リストの表示対象候補となる地物型の名前がここに示されます．

Proxy
リストの対象になる地物型の proxy の名前がここに示されます．

Proxy value list （選択可能）
Proxyの値の一覧がここに示されます．このリストが，地物の一次元地図です．このリスト上で，地物を選択すると，その地物の属性値を表示することができます．

Attributes （選択可能）
地物属性の一覧から，一つ選択すると，その属性の値を示すページが現れます．

Buttons

open (Kit)
このボタンを押すと，リスト表示の対象となる kit がオープンします．

open (List schema)
このボタンを押すと，リスト表示の規則を示すリストスキーマ がオープンします．

Proxy value list
この一覧から地物インスタンスを選択すると，その他の地物属性の表示が可能になります．

Attributes
この一覧から地物属性を選択すると，info. pageが表示されます．幾何属性の場合は，geometry pageが表示されます．
