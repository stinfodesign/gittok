Classpublic class StyledMapType
Implementsflash.events.IEventDispatcher, IStyledMapType

Class StyledMapType defines a styled map type for the map.


Example
Set one styled map type, then change to another. 
   import com.google.maps.StyledMapType;
   import com.google.maps.StyledMapTypeOptions;
   import com.google.maps.styles.MapTypeStyleElementType;
   import com.google.maps.styles.MapTypeStyleFeatureType;
   import com.google.maps.styles.MapTypeStyleRule;

   var options:StyledMapTypeOptions = new StyledMapTypeOptions({
       name: 'Styled map',
       alt: 'Style',
       minResolution: 2,
       maxResolution: 12
   });

   var originalStyles:Array = [
     new MapTypeStyle(MapTypeStyleFeatureType.ALL,
                      MapTypeStyleElementType.ALL,
                      [MapTypeStyleRule.saturation(-80)]),
     new MapTypeStyle(MapTypeStyleFeatureType.ROAD,
                      MapTypeStyleElementType.GEOMETRY,
                      [MapTypeStyleRule.saturation(70),
                       MapTypeStyleRule.hue(0xff0000)])
     ];
   var styledMapType:StyledMapType =
       new StyledMapType(originalStyles, options);
   map.addMapType(styledMapType);
   map.SetMapType(styledMapType);

     :

   var updatedStyles:Array = [
     new MapTypeStyle(MapTypeStyleFeatureType.ALL,
                      MapTypeStyleElementType.ALL,
                      [MapTypeStyleRule.lightness(-20)])
     ];
   styledMapType.setStyle(updatedStyles);

Example
Setting the style using an array of MapTypeStyle instance. 
     styledMapType.setStyle([
         new MapTypeStyle(MapTypeStyleFeatureType.ALL,
                          MapTypeStyleElementType.ALL,
                          [MapTypeStyleRule.saturation(-20)]),
         new MapTypeStyle(MapTypeStyleFeatureType.ROAD,
                          MapTypeStyleElementType.GEOMETRY,
                          [MapTypeStyleRule.hue(0xff0000)])]);
         ]);

Setting the style using an array of objects. 
     styledMapType.setStyle([
         {elementType: 'all',
          featureType: 'all',
          stylers: [{saturation: -20}]},
         {elementType: 'geometry',
          featureType: 'road',
          stylers: [{hue: 0xff0000}]}]);
