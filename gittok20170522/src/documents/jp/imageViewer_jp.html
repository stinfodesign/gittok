Image Viewer
画像表示

 

はじめに

このページは，地物を画像として示す属性の表示を行うためにあります．gittok では，地物は，画像を保存している場所(URL) を属性とします．Image Viewer は，その保存場所から画像を取得して，表示します．

 

Image Viewer の操作画面

図1. Image Viewer

FIELDS

Attributes (selectable)
表示対象になる画像属性を選択します．

List
属性になっている画像ファイル名の一覧が表示されます．

Image Panel
このパネルに画像が表示されます．

BUTTONS

fix / pan
画像の移動を可能にするときは pan，そうでないときは fix にする．

>
このボタンを押すと画像が縮小される．

<
このボタンを押すと，画像が拡大される．

-
このボタンを押すと，表示される画面縮尺が，初期化される．

日本語
今あなたが読んでいるドキュメントが表示されます．
