Image Viewer

 

Introduction

This page displays image attributes of a feature. Image viewer brings images from "images" sub-folder in the project folder.

 

Image Viewer

Figure 1. Image Viewer

FIELDS

attributes (selectable)
A user can select an attribute name on this list.

List (selectable)
This field displays a list of image file names as attributes of a feature.

Image Panel
This panel displays an image selected from the list.

BUTTONS

fix / pan
An image is translated before the button is clicked to make it 'pan'.

>
An image is smaller by clicking this button.

<
An image is larger by clicking this button.

-
The scale and position are initialized by chicking this button.
