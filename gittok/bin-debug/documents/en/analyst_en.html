Analyst

Analyst enables to execute operations included in feature or association. Operation is a method or function to return data as a result of processing by receiving arguments as input data. In the context of geospatial information sciences, operations using position on the earth is called spatial analysis. This is the reason that the name of this page is Analyst.

This text explains the procedure of the execution of an operation, fields and buttons in Analyst page, the mechanism of operation, and how to buid in operations in gittok.

Operation procedure

1. Base map and geometries appear in Instance view panel by opening a kit file.
2. "Feature" or "Association" tab is selected in order to execute operations.

In case of Feature

1. Radio button of geometry type that is a proxy of the feature is selected by selecting feature type at Feature type list.
2. Instance id is shown in Instance ID field by selecting feature instance on Instance view panel. Simultaneously, a name and return data type of the operation are displayed on Operation grid.
3. Operation is executed by clicking "execute" button after the selection of operation at Operation grid.
4. The result of operation is displayed in Return attribute value field. If the return data type is simple, the value is shown in the value field. While if it is complex, an id of the return value is shown in the id field.

In case of Association

1. The names of feature types are displayed in From and To fields and the list of operations are shown in Operation grid by selecting association type at Association type list.
2. Association instance is selected at Association instance ID list.
3. Instance ids are displayed in From and To boxes.
4. If it is necessary to confirm the selection, click show button. Then blue triangles (From) and red triangles (to) are displayed in the panel.
5. Click "execute" button to execute the operation.
6. he result of operation is displayed in Return attribute value field. If the return data type is simple, the value is shown in the value field. While if it is complex, an id of the return value is shown in the id field.

Display of the result

If the return value is geometry, the result is displayed in Instance view panel.

Analyst

Feature

Figure 1. Page of feature operation

Fields

Kit
The name of the kit is shown in this field.

Operation (selectable)
The names and data types of return value for operations are listed. operation shall be selected at this grid before execution.

Return attribute value
The result of operation is shown. If the data type of a return value is simple, the value is shown. While if it is complex, the id of the return value is shown.

Instance view panel
Geometries and symbols of feature and association instances are displayed on the base map.

Feature type list (selectable)
Feature type which includes the operation can be selected from the list.

Instance ID
Instance ID selected at Instance view panel is shown in this field.

Buttons

open (Kit)
Kit can be opened by clicking this button.

save (Kit)
Kit with the result of operations can be saved by using this button.

execute
An operation is executed by clicking this button.

>
Map scale is smaller by clicking this button.

<
Map scale is larger by clicking this button.

-
Initialize the map scale by clicking this button.

slider
Transparency of the base map changes by dragging slider button.
