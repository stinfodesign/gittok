Acquisition Manager

Introduction

Acquisition Manager allows you to input geospatial data as instances of feature or association in accordance with the application schema which you designed. You can save geospatial dataset called "kit" as a result of acquisition. First, this section explains instance model that is a rule how to describe a kit as a set of instances. Next, the procedure of acquisition will be explained. Then, We will learn how to use Acquisition Manager page.

Instance Model

What is an instance model?
Instance is data constructed in accordance with the application schema. Instance model is a specification to show how to make instances (See Figure 1.).
