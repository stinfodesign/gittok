Analyst

Introduction

Analyst performs operations of features or associations specified in the application schema. Operation inputs data called an argument, processes in order to get information, and outputs information called as return value. Gittok puts return value in the attribute included in a feature. In other words, you need the definition of attribute for return value, if you will define operation.
The term "Spatial Analysis" is an alternative term of operation. This is the reason that the name of this page is "Analyst".

Now we will learn the procedure of operation in gittok. Next we will learn how to interact with Analyst. Finally, you can study the method to install your original operation in gittok. Gittok is an open source software. We recommend you to develop your own applications and apply them in your gittok!
